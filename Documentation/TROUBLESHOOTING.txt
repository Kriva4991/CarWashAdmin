CarWash Admin Pro — Версия 3.0

🚨 КРИТИЧЕСКИЕ ПРОБЛЕМЫ

❌ Приложение не запускается вообще
Симптомы:
• Двойной клик на .exe — ничего не происходит
• Или мгновенное закрытие без ошибок

Решение:
1. Проверьте наличие .NET и Visual C++:
   • Скачайте с сайта Microsoft
2. Запустите от имени администратора
3. Проверьте антивирус (может блокировать .exe)
4. Попробуйте запустить из консоли:
   cd dist/CarwashAdmin
   CarwashAdmin.exe
   ← Смотрите ошибки в консоли

❌ Ошибка: "Qt platform plugin 'windows' could not be loaded"

Причина: Проблемы с PyQt6

Решение:
1. Переустановите PyQt6:
   pip uninstall PyQt6
   pip install PyQt6==6.4.2
2. Пересоберите .exe:
   pyinstaller --clean src/main.py
3. Проверьте папку _internal — должны быть файлы Qt

❌ Ошибка базы данных: "database disk image is malformed"

Причина: Повреждение carwash.db

Решение:
1. Найдите последнюю резервную копию
2. Восстановите:
   • Закройте приложение
   • Удалите data/carwash.db
   • Скопируйте бэкап с именем carwash.db
3. Если бэкапов нет — обратитесь в поддержку

Профилактика:
• Делайте бэкапы ежедневно
• Храните копии на другом диске/в облаке
• Не выключайте ПК во время записи в БД

❌ Забыли пароль администратора

Вариант 1: Через поддержку (рекомендуется)
• Напишите в поддержку с подтверждением личности
• Получите инструкцию по сбросу

Вариант 2: Самостоятельно (для продвинутых)
1. Установите DB Browser for SQLite
2. Откройте data/carwash.db
3. Выполните запрос:
   UPDATE users SET password_hash = '<новый_хэш>' 
   WHERE username = 'admin'
4. Сгенерируйте хэш онлайн:
   • Алгоритм: bcrypt, rounds: 12

⚠️ ПРОБЛЕМЫ С ИНТЕРФЕЙСОМ

❌ Элементы не видны / сливаются с фоном

Причина: Конфликт тем Windows и PyQt

Решение:
• В main.py уже добавлена принудительная светлая тема
• Если проблема осталась — пересоберите .exe
• Проверьте масштабирование Windows (100% рекомендовано)

❌ Графики не отображаются / пустые

Причина: Нет данных за выбранный период

Решение:
1. Проверьте фильтр периода (Сегодня/Неделя/Месяц)
2. Убедитесь что есть заказы с статусом не "Отменено"
3. Проверьте дату создания заказов в БД

❌ Экспорт в PDF: квадраты вместо текста

Причина: Отсутствует шрифт с кириллицей

Решение:
1. Установите DejaVu Sans
2. Или используйте Arial
3. Или используйте экспорт в CSV (работает всегда)

📊 ПРОБЛЕМЫ С ДАННЫМИ

❌ В экспорте пусто: услуги, сумма, телефон

Причина: Старая версия кода экспорта

Решение:
• Убедитесь что используется версия 3.0
• Проверьте что в БД есть данные в order_items
• Пересоберите .exe после обновлений кода

❌ Дубликаты клиентов в базе

Причина: Клиент добавлен несколько раз с разным написанием

Решение:
1. Используйте поиск перед созданием заказа
2. Если дубликаты есть — объедините вручную:
   • Экспортируйте базу в CSV
   • Удалите дубли в Excel
   • Импортируйте обратно (через поддержку)

❌ Неправильная сумма в заказе

Причина: Редактирование цены + несколько услуг

Решение:
• Проверьте итоговую сумму перед сохранением
• Сумма = Σ (цена_услуги × количество) с учётом скидок
• При необходимости отредактируйте вручную в форме

🔄 ПЕРЕУСТАНОВКА / ОБНОВЛЕНИЕ

❌ Как обновить до новой версии

1. Сделайте резервную копию:
   • Скопируйте data/carwash.db в безопасное место
2. Скачайте новую версию от разработчика
3. Замените файлы в папке программы
4. Запустите — миграции применятся автоматически

❌ Как откатиться на старую версию

1. Сделайте бэкап текущей базы
2. Удалите новую версию
3. Установите старую версию
4. Восстановите базу из бэкапа (если совместима)
