🛠️ КАК ДЕЛАТЬ БЭКАП (РУЧНОЙ)

Способ 1: Через Проводник
1. Закройте CarWash Admin полностью
2. Откройте папку программы
3. Скопируйте data/carwash.db
4. Вставьте в папку бэкапов
5. Переименуйте с датой:
   carwash_backup_2026-03-27.db

Способ 2: Через командную строку
cmd:
cd C:\Path\To\CarwashAdmin
copy data\carwash.db backups\carwash_backup_%date:~6,4%-%date:~3,2%-%date:~0,2%.db

☁️ АВТОМАТИЗАЦИЯ (ПЛАНИРОВЩИК ЗАДАЧ)

1. Создайте .bat файл
2. Откройте "Планировщик заданий" (taskschd.msc)
3. Создайте задачу:
   • Триггер: Ежедневно, 23:00
   • Действие: Запустить ваш .bat файл
   • Условия: Запускать только от имени администратора

🗄️ ГДЕ ХРАНИТЬ БЭКАПЫ

Рекомендуемая стратегия 3-2-1:

3 копии данных
2 разных типа носителей
1 копия вне площадки

Пример:
✓ Локально: D:\Backups\Carwash\
✓ Внешний диск: Е:\Архив\
✓ Облако: Яндекс.Диск / Google Drive

🔄 КАК ВОССТАНОВИТЬ

Шаг 1: Подготовка
1. Закройте CarWash Admin полностью
2. Найдите актуальный бэкап

Шаг 2: Восстановление
Исходная папка:
C:\CarwashAdmin\data\carwash.db  ← УДАЛИТЬ или переименовать

Бэкап:
D:\Backups\carwash_backup_2026-03-27.db  ← ПЕРЕИМЕНОВАТЬ

Действие:
1. Переименуйте carwash.db → carwash_old.db
2. Скопируйте бэкап в data/
3. Переименуйте бэкап в carwash.db
4. Запустите программу

Шаг 3: Проверка
✓ Откройте программу
✓ Проверьте последние заказы
✓ Убедитесь что клиенты на месте

⚠️ ЧАСТЫЕ ОШИБКИ

❌ Бэкап открытой базы
• Решение: Всегда закрывайте программу перед копированием

❌ Бэкап без даты в имени
• Решение: Всегда добавляйте дату: backup_YYYY-MM-DD.db

❌ Хранение только локально
• Решение: Используйте облако или внешний диск

❌ Редкие бэкапы
• Решение: Настройте автоматизацию

🔍 ПРОВЕРКА ЦЕЛОСТНОСТИ БЭКАПА

Периодически проверяйте бэкапы:
1. Скопируйте бэкап в тестовую папку
2. Переименуйте в carwash.db
3. Запустите программу с этой папкой
4. Убедитесь что данные отображаются
5. Удалите тестовую папку

📞 ПРОБЛЕМЫ С ВОССТАНОВЛЕНИЕМ?

Если восстановление не работает:
1. Не паникуйте — данные скорее всего целы
2. Не перезаписывайте оригинальный файл
3. Напишите в поддержку с:
   • Версией программы
   • Ошибкой при восстановлении
   • Размером файла бэкапа
