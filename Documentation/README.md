ВВЕДЕНИЕ
CarWash Admin Pro — программа для автоматизации работы автомойки.
Что умеет:
Учёт заказов (несколько услуг в одном заказе)
База клиентов с историей посещений
Справочник услуг с ценами
Отчёты по выручке и статистике
Управление сменами
Автоматические резервные копии
Экспорт данных в Excel/PDF
================================================================
СИСТЕМНЫЕ ТРЕБОВАНИЯ
ОС: Windows 10/11
Процессор: 1.5 ГГц и выше
ОЗУ: 4 ГБ
Место на диске: 500 МБ
Интернет: Не требуется (работает офлайн)
================================================================
УСТАНОВКА
Шаг 1: Запустите установщик
CarWashAdmin_Setup.exe
Шаг 2: Следуйте инструкциям
Выберите папку установки (по умолчанию: C:\Program Files\CarWashAdmin)
Создайте ярлык на рабочем столе
Шаг 3: Готово!
Программа установлена. Можно запускать.
================================================================
ПЕРВЫЙ ЗАПУСК
Откройте программу (двойной клик на ярлык)
Войдите в систему:
Логин: admin
Пароль: admin123
Смените пароль!
Настройки → Безопасность → смените пароль
Активируйте лицензию
Настройки → Лицензия → введите ключ
Настройте бэкапы
Настройки → Резервное копирование → включите автобэкап
================================================================
ВХОД В СИСТЕМУ
Поля для входа:
Пользователь: Ваше имя (например, admin)
Пароль: Ваш пароль (по умолчанию: admin123)
Важно! После первого входа обязательно смените пароль!
================================================================
ОСНОВНОЙ ИНТЕРФЕЙС
Программа состоит из 5 вкладок:
[📋] Текущие заказы — Создание и управление заказами
[⚙️] Услуги — Справочник услуг и цен
[👤] Клиенты — База клиентов с историей
[📊] Отчёты — Статистика и аналитика
[🔧] Настройки — Конфигурация программы
================================================================
РАБОТА С ЗАКАЗАМИ
СОЗДАНИЕ ЗАКАЗА:
Нажмите "➕ Новая заявка" или клавиши Ctrl+N
Заполните данные:
Гос. номер (обязательно)
Марка/модель авто
Телефон клиента
Класс авто (влияет на цену)
Выберите услуги (можно несколько)
Нажмите "💾 Сохранить"
СТАТУСЫ ЗАКАЗА:
🟡 В очереди — Заказ принят, ждёт мойки
🔵 В работе — Автомобиль моется
🟢 Готово — Мойка завершена
Как сменить статус:
Двойной клик по заказу в таблице
Или выделите заказ и нажмите Enter
РЕДАКТИРОВАНИЕ ЗАКАЗА:
Выделите заказ в таблице
Нажмите "✏️" или клавишу F2
Внесите изменения
Нажмите "💾 Сохранить"
УДАЛЕНИЕ ЗАКАЗА:
Выделите заказ в таблице
Нажмите "🗑️" или клавишу Delete
Подтвердите удаление
ПОИСК ЗАКАЗА:
В поле "🔍 Поиск" введите:
Гос. номер
Телефон клиента
ID заказа
Название услуги
Таблица автоматически отфильтруется.
================================================================
УСЛУГИ
ДОБАВЛЕНИЕ УСЛУГИ:
Перейдите во вкладку "⚙️ Услуги"
Нажмите "➕ Добавить услугу"
Заполните:
Название (например, "Комплексная мойка")
Цена (базовая)
Длительность в минутах (опционально)
Нажмите "💾 Сохранить"
КАТЕГОРИИ УСЛУГ:
🟦 Мойка — Комплекс, Экспресс, Ручная
🟧 Дополнительные — Коврики, Чернение, Антидождь
🟪 Покрытия — Воск, Кварц, Керамика
РЕДАКТИРОВАНИЕ УСЛУГИ:
Нажмите "✏️" на карточке услуги → внесите изменения → сохраните.
УДАЛЕНИЕ УСЛУГИ:
Нажмите "🗑️" на карточке услуги → подтвердите удаление.
================================================================
КЛИЕНТЫ
ПРОСМОТР КЛИЕНТОВ:
Во вкладке "👤 Клиенты" отображаются все клиенты:
🟢 Постоянный — 3+ посещений
🔵 Новый — 1-2 посещения
ИСТОРИЯ ПОСЕЩЕНИЙ:
Кликните на карточку клиента — увидите историю всех заказов.
ПОИСК КЛИЕНТА:
Введите в поле поиска:
Гос. номер
Телефон
Имя или фамилию
================================================================
ОТЧЁТЫ
ДОСТУПНЫЕ ОТЧЁТЫ:
Заказы сегодня — Количество заказов за день
Выручка — Сумма за выбранный период
В работе — Текущие активные заказы
Топ услуг — Самые популярные услуги
ФИЛЬТРЫ:
Период: сегодня, неделя, месяц, произвольный
Статус: все, выполненные, в работе
ЭКСПОРТ:
Нажмите "📥 Экспорт CSV" или "📄 Экспорт PDF" для сохранения отчёта.
================================================================
НАСТРОЙКИ
БЕЗОПАСНОСТЬ:
Текущий пароль: Ваш действующий пароль
Новый пароль: Новый пароль (минимум 6 символов)
Подтверждение: Повторите новый пароль
Важно! Запомните новый пароль!
ИНФОРМАЦИЯ О КОМПАНИИ:
Заполните данные для чеков и отчётов:
Название компании
Телефон
Адрес
Сайт
Логотип
РЕЗЕРВНОЕ КОПИРОВАНИЕ:
Включить автобэкап: Галочка для активации
Папка для бэкапов: Куда сохранять копии
День недели: Когда делать бэкап
Время: Во сколько (формат ЧЧ:ММ)
Рекомендация: Включите автобэкап на каждый день в 23:00
ЛИЦЕНЗИЯ:
Типы лицензий:

Бессрочный: Навсегда, формат ключа: CW-LIFE-XXXX-XXXX-CCCC
Для активации:
Введите ключ в поле
Нажмите "✅ Активировать"
Готово!

ГОРЯЧИЕ КЛАВИШИ
ГЛОБАЛЬНЫЕ:
Ctrl+N — Новый заказ
Ctrl+S — Сохранить
F5 — Обновить данные
Ctrl+F — Поиск
F12 — Управление сменами
Esc — Отмена / Закрыть
НАВИГАЦИЯ ПО ВКЛАДКАМ:
Ctrl+1 — 📋 Заказы
Ctrl+2 — ⚙️ Услуги
Ctrl+3 — 👤 Клиенты
Ctrl+4 — 📊 Отчёты
Ctrl+5 — 🔧 Настройки
В ТАБЛИЦЕ ЗАКАЗОВ:
F2 — Редактировать заказ
Delete — Удалить заказ
Enter — Сменить статус

РЕЗЕРВНОЕ КОПИРОВАНИЕ
АВТОМАТИЧЕСКИЙ БЭКАП:
Откройте Настройки → Резервное копирование
Включите "🔄 Включить автоматический бэкап"
Укажите папку (например, D:\Backups\Carwash)
Выберите день и время
Нажмите "💾 Сохранить настройки"
РУЧНОЙ БЭКАП:
Откройте Настройки → Резервное копирование
Нажмите "💾 Создать бэкап сейчас"
Дождитесь сообщения об успехе
ВОССТАНОВЛЕНИЕ ИЗ БЭКАПА:
Внимание! Делайте только при потере данных!
Закройте программу
Скопируйте файл бэкапа (carwash_backup_*.db) в папку data\
Переименуйте в carwash.db
Запустите программу
================================================================
ЛИЦЕНЗИЯ
ПРОВЕРКА СТАТУСА:
Откройте Настройки → Лицензия
Статусы:
🟢 Бессрочная лицензия — Всё работает без ограничений
🟡 Пробный период — Работает, но есть срок (показывает сколько дней осталось)
🔴 Срок истёк — Требуется продление
ПРОДЛЕНИЕ ЛИЦЕНЗИИ:
Свяжитесь с поддержкой (контакты ниже)
Сообщите что нужна новая лицензия
Получите новый ключ
Введите ключ в программе → Активировать
ЧТО ДЕЛАТЬ ЕСЛИ ИСТЕКЛА:
При запуске появится окно с уведомлением
Нажмите "Да" — откроются настройки
Введите новый ключ
Программа разблокируется

ЧАСТЫЕ ВОПРОСЫ
В: Как добавить нового пользователя?
О: Сейчас поддерживается один пользователь "admin". Для добавления других обратитесь в поддержку.
В: Где хранятся данные?
О: В папке data\carwash.db (внутри папки программы).
В: Можно ли работать без интернета?
О: Да! Программа работает полностью офлайн.
В: Как перенести данные на другой компьютер?
О:
Скопируйте файл data\carwash.db
Установите программу на новом ПК
Замените файл data\carwash.db
Активируйте лицензию заново
В: Что делать если программа не запускается?
О:
Проверьте что у вас Windows 10/11
Запустите от имени администратора
Проверьте антивирус (может блокировать)
Обратитесь в поддержку
В: Как изменить пароль?
О: Настройки → Безопасность → введите текущий и новый пароль → "Сменить пароль"
В: Можно ли экспортировать данные?
О: Да! Вкладка Отчёты → Экспорт CSV или Экспорт PDF
