## ВВЕДЕНИЕ

**CarWash Admin Pro** — профессиональная программа для автоматизации работы автомойки.

### Возможности:
- ✅ Учёт заказов с группировкой по статусам (очередь / в работе / готово)
- ✅ Несколько услуг в одном заказе
- ✅ Редактирование цен (скидки на лету)
- ✅ База клиентов с историей посещений и статистикой
- ✅ Учёт расходных материалов (остатки, списания)
- ✅ Система ролей (админ / менеджер / мойщик)
- ✅ Журнал аудита (кто и что изменил)
- ✅ Отчёты с экспортом в Excel (XLSX)
- ✅ Поддержка русского и английского языков
- ✅ Автоматическая проверка обновлений

## УСТАНОВКА

### Портативная версия (рекомендуется):
1. Скопируйте папку `CarWashAdmin` в любое удобное место (например, `C:\Program Files\CarWashAdmin`)
2. Запустите `CarWashAdmin.exe`
3. Готово!

### Установщик:
1. Запустите `CarWashAdmin_Setup_3.0.0.exe`
2. Следуйте инструкциям мастера установки
3. Выберите папку для установки
4. Создайте ярлык на рабочем столе (опционально)
5. Нажмите «Установить»

## ПЕРВЫЙ ЗАПУСК

1. Запустите программу (двойной клик по ярлыку или `.exe`)
2. Войдите в систему:
   - **Логин:** `admin`
   - **Пароль:** `admin123`
3. Активируйте лицензию:
   - Настройки → Лицензия → введите ключ → «Активировать»
4. Настройте компанию:
   - Настройки → Информация о компании → заполните данные → «Сохранить»
5. Настройте автообновление:
   - Настройки → Автообновление → включите проверку

## ПОЛЬЗОВАТЕЛИ И РОЛИ

Программа поддерживает три роли с разными правами доступа.

### 👑 Администратор (полный доступ):
- Создание, редактирование и удаление заказов
- Управление клиентами и услуг
- Просмотр всех отчётов и экспорт
- Управление расходными материалами
- Управление пользователями и ролями
- Просмотр журнала аудита
- Активация лицензии и настройка программы

### 👔 Менеджер (ограниченный доступ):
- Создание и редактирование заказов
- Изменение статусов заказов
- Просмотр и редактирование клиентов
- Просмотр услуг
- Просмотр отчётов и экспорт в Excel
- Просмотр расходных материалов

### 🧽 Мойщик (минимальный доступ):
- Просмотр заказов
- Изменение статуса заказа (очередь → в работе → готово)
- Просмотр клиентов

### Управление пользователями (только для админа):

**Как открыть:**
Настройки → Пользователи системы → Управление пользователями

**Доступные действия:**
- ➕ **Добавить** — создание нового пользователя (логин, пароль, роль)
- ✏️ **Редактировать** — изменение роли пользователя
- 🔑 **Сбросить пароль** — установка нового пароля
- 🔄 **Активировать/Блокировать** — временное отключение пользователя
- 🗑️ **Удалить** — полное удаление пользователя

### Журнал аудита (только для админа):

**Как открыть:**
Настройки → Пользователи системы → Журнал аудита

**Что отслеживается:**
- Вход и выход из системы
- Создание/редактирование/удаление заказов
- Изменение статусов
- Управление пользователями
- Попытки доступа без прав

## РАБОТА С ЗАКАЗАМИ

### Создание заказа:

1. Нажмите **«➕ Новый заказ»** или клавиши `Ctrl+N`
2. Заполните данные:
   - **Гос. номер** (обязательно) — например, `А123БВ77`
   - **Марка/модель** — например, `Toyota Camry`
   - **Телефон клиента** — для связи и истории
   - **Класс авто** — влияет на коэффициент цены
3. Выберите услуги (можно несколько):
   - Отметьте галочкой нужные услуги
   - При необходимости измените цену и количество
4. Нажмите **«💾 Сохранить»**

### Статусы заказа:

| Статус | Описание | Действие |
|--------|----------|----------|
| 🟡 В очереди | Заказ принят, ждёт мойки | Нажмите **«▶ В работу»** |
| 🔵 В работе | Автомобиль моется | Нажмите **«✓ Готово»** |
| 🟢 Готово | Мойка завершена | — |

> 💡 **Подсказка:** Заказы группируются по статусам. При переводе в «Готово» карточка автоматически перемещается в соответствующую группу.

### Группировка заказов:

Заказы отображаются в трёх группах:
- 🟡 **В очереди** — новые заказы
- 🔵 **В работе** — заказы в процессе выполнения
- 🟢 **Готово** — завершённые заказы

Пустые группы показывают заглушку «📭 Нет заказов...».

### Действия с заказом:

На каждой карточке заказа доступны кнопки:
- **▶ В работу / ✓ Готово** — смена статуса (в зависимости от текущего)
- **✏️ Изменить** — открыть форму редактирования
- **🗑️ Удалить** — удалить заказ (только для админа)

> ⚠️ **Важно:** Двойной клик по карточке заказа также открывает форму редактирования.

### Поиск заказов:

В поле **«🔍 Поиск»** можно ввести:
- Гос. номер
- Телефон клиента
- ID заказа
- Марку авто

Таблица фильтруется автоматически при вводе.

### Просмотр всех заказов:

По умолчанию показываются только **сегодняшние заказы**.  
Чтобы увидеть историю за всё время:
- Отметьте галочку **«📅 Все заказы»**
- Будет показано до 200 последних заказов

> 💡 Для полной истории используйте вкладку **«📊 Отчёты»**.

## УСЛУГИ

### Просмотр услуг:

Вкладка **«⚙️ Услуги»** отображает все услуги в виде карточек.

**Категории услуг:**
- 🧽 **Мойка** — комплексная, кузов, салон
- ➕ **Дополнительные** — коврики, чернение резины, антидождь
- ✨ **Покрытия** — воск, кварц, керамика

### Поиск и фильтрация:

- **🔍 Поиск** — поиск по названию услуги
- **📂 Категория** — фильтр по категориям (Все / Мойка / Дополнительные / Покрытия)

### Добавление услуги (только админ):

1. Нажмите **«➕ Добавить услугу»**
2. Заполните:
   - **Название** — например, «Полировка кузова»
   - **Цена** — базовая стоимость
   - **Время** — длительность в минутах
3. Нажмите **«💾 Сохранить»**

### Редактирование услуги (только админ):

- Нажмите **«✏️»** на карточке услуги
- Внесите изменения
- Нажмите **«💾 Сохранить»**

### Удаление услуги (только админ):

- Нажмите **«🗑️»** на карточке услуги
- Подтвердите удаление

## КЛИЕНТЫ

### Просмотр клиентов:

Вкладка **«👤 Клиенты»** отображает всех клиентов в виде карточек.

**Цветовая индикация по активности:**
- 🟢 **Зелёный** — VIP (5+ визитов)
- 🟠 **Оранжевый** — Постоянный (2-4 визита)
- 🔵 **Синий** — Новый (1 визит)

**На карточке отображается:**
- Гос. номер и марка авто
- Телефон
- Количество визитов
- Общая сумма потраченных средств
- Дата последнего визита
- Комментарий (если есть)

### Поиск клиентов:

В поле **«🔍 Поиск»** можно искать по:
- Гос. номеру
- Телефону
- Марке авто

### Пагинация:

Если клиентов больше 50, внизу появляются кнопки:
- **← Назад / Вперёд →** — переключение страниц
- Выпадающий список — быстрый переход к нужной странице

### Синхронизация:

Кнопка **«🔄 Синхронизировать»** создаёт клиентов из существующих заказов.  
Полезно при первом запуске или если клиенты не создавались автоматически.

### История заказов:

- Двойной клик по карточке клиента
- Или кнопка **«📋 История»**

Открывается окно с полной историей заказов клиента и статистикой.

### Комментарий:

- Кнопка **«✏️ Комментарий»** — добавить или изменить заметку о клиенте
- Например: «VIP, любит кофе», «Скидка 10%», «Торопится всегда»

### Экспорт клиентов:

Кнопка **«📊 Excel»** — экспорт всей базы клиентов в Excel с форматированием.

## РАСХОДНЫЕ МАТЕРИАЛЫ

Вкладка **«🧴 Расходники»** позволяет вести учёт материалов.

### Просмотр:

- Список всех расходников с остатками
- **Цветовая индикация:**
  - 🟢 Норма — остаток выше минимального
  - 🟡 Мало — остаток равен или ниже минимального
  - 🔴 Закончился — остаток 0

### Статистика:

В верхней части отображается:
- Всего позиций
- Количество позиций с низким запасом
- Общая стоимость остатков

### Предупреждение:

Если есть материалы с низким запасом, показывается жёлтое предупреждение с перечислением.

### Операции (только админ и менеджер):

| Кнопка | Действие |
|--------|----------|
| ➕ Добавить | Создание нового расходника |
| 📦 Пополнить | Увеличение остатка |
| 📉 Списать | Уменьшение остатка (с указанием причины) |
| ✏️ Редактировать | Изменение названия, ед. измерения, мин. запаса, цены |
| 📋 История | Просмотр всех списаний по выбранному расходнику |
| 🗑️ Удалить | Удаление расходника |

## ОТЧЁТЫ

Вкладка **«📊 Отчёты»** предоставляет аналитику по заказам.

### Фильтры периода:

- **Сегодня** — заказы за текущий день
- **Неделя** — за последние 7 дней
- **Месяц** — за текущий месяц
- **Произвольно** — выбор дат вручную

### Метрики:

В верхней части отображаются карточки:
- 💰 **Выручка** — общая сумма за период
- 📋 **Заказы** — количество заказов
- 🧮 **Средний чек** — выручка / количество заказов
- 🔧 **Услуг** — количество оказанных услуг

### Графики:

- **Выручка по дням** — столбчатая диаграмма
- **Топ услуг** — круговая диаграмма самых популярных услуг

### Детализация:

Таблица со списком всех заказов за выбранный период:
- Дата и время
- Гос. номер
- Услуги
- Сумма
- Статус
- Способ оплаты

### Экспорт в Excel:

Кнопка **«📊 Excel»** создаёт отчёт с:
- Заголовком и периодом
- Статистикой (всего заказов, выручка)
- Детализацией по способам оплаты (наличные, карта, перевод, СБП)
- Таблицей заказов с цветовым выделением по статусам
- Итоговой суммой

## РЕЗЕРВНОЕ КОПИРОВАНИЕ

### Автоматический бэкап:

1. Откройте **Настройки → Резервное копирование**
2. Включите «🔄 Включить автоматический бэкап»
3. Укажите папку для сохранения
4. Выберите день недели и время
5. Нажмите «💾 Сохранить настройки»

### Ручной бэкап:

1. Откройте **Настройки → Резервное копирование**
2. Нажмите «💾 Создать бэкап сейчас»
3. Дождитесь сообщения об успехе

### Восстановление из бэкапа:

1. Закройте программу
2. Найдите файл бэкапа (например, `carwash_backup_2026-04-16.db`)
3. Скопируйте его в папку `data\` внутри папки программы
4. Переименуйте в `carwash.db` (заменив существующий)
5. Запустите программу

> ⚠️ **Внимание!** Восстановление перезаписывает текущие данные. Делайте бэкап перед восстановлением!

## ЛИЦЕНЗИЯ

### Типы лицензий:

| Тип | Срок | Формат ключа |
|-----|------|--------------|
| Пробная | 30 дней | `CW-30-XXXX-XXXX-CCCC` |
| Бессрочная | Навсегда | `CW-LIFE-XXXX-XXXX-CCCC` |

### Активация:

1. Откройте **Настройки → Лицензия**
2. Введите ключ в поле «Ключ активации»
3. Нажмите «✅ Активировать»

### Статусы лицензии:

| Статус | Описание |
|--------|----------|
| 🟢 Бессрочная | Работает без ограничений |
| 🟡 Пробный период | Работает, осталось N дней |
| 🔴 Истекла | Требуется продление |

### Что делать если истекла:

1. Свяжитесь с поддержкой (контакты в `CONTACTS.txt`)
2. Получите новый ключ
3. Введите ключ в программе и активируйте

> 🔒 **Примечание:** Ключ отображается в замаскированном виде для безопасности.

## ЧАСТЫЕ ВОПРОСЫ

### В: Как добавить нового пользователя?
**О:** Настройки → Пользователи системы → Управление пользователями → «➕ Добавить». Доступно только для админа.

### В: Как сменить пароль?
**О:** Настройки → Пользователи системы → Управление пользователями → выберите пользователя → «🔑 Сбросить пароль».

### В: Где хранятся данные?
**О:** В папке `data\carwash.db` внутри папки программы.

### В: Можно ли работать без интернета?
**О:** Да! Программа работает полностью офлайн. Интернет нужен только для активации лицензии и проверки обновлений.

### В: Как перенести данные на другой компьютер?
**О:**
1. Скопируйте файл `data\carwash.db`
2. Установите программу на новом ПК
3. Замените файл `data\carwash.db`
4. Активируйте лицензию заново

### В: Что делать если программа не запускается?
**О:**
1. Проверьте, что у вас Windows 10/11 (64-bit)
2. Запустите от имени администратора
3. Проверьте антивирус (может блокировать)
4. Обратитесь в поддержку

### В: Как переключить язык на английский?
**О:** Настройки → Оформление и язык → выберите «English» → перезапустите программу.

### В: Как посмотреть кто и что изменил?
**О:** Настройки → Пользователи системы → Журнал аудита. Доступно только для админа.

### В: Почему не видно старых заказов?
**О:** По умолчанию показываются только сегодняшние заказы. Отметьте галочку «📅 Все заказы» или используйте вкладку «Отчёты» для просмотра истории.
